# Interdisciplinar Web - Laboratório de Prótese

Sistema web desenvolvido em ASP.NET Core MVC para gerenciamento de pedidos e serviços de um laboratório de prótese dentária.

## Objetivo

O projeto tem como foco auxiliar no controle de:

- Cadastro de usuários
- Serviços do laboratório
- Carrinho de pedidos
- Controle de pedidos
- Quantidade de itens
- Fluxo de autenticação

## Tecnologias Utilizadas

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- Bootstrap
- HTML/CSS/JavaScript
